# Distilled Capabilities

See the repository-level canonical reference at
`skills/fairy-tale/references/capabilities.md` when developing in this repo.

When installed as a plugin, use these core process capabilities:

- autonomous long-task execution with explicit budgets,
- codebase-wide engineering with scoped checkpoints,
- high-end analytical work with evidence maps,
- vision-grounded reconstruction with rendered-output validation,
- defensive security review with authorization and patch-first output,
- workflow self-improvement through config/skill/command inventory.
